# 전국 공보 지도

'공보’란 국가기관과 지방자치단체가 대국민에 정부의 소식을 알려주는 소식지를 말합니다. 공보는 누구나 찾아볼 수 있도록 제공되고 있으며, 시민단체와 언론이 행정 영역을 감시할 때는 공공 감시 도구가 되기도 합니다. 그렇다면, 인터넷 시대에 '전자 공보'는 우리 시민의 곁에 얼마나 더 가까워졌을까요?
